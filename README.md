# LyricSense_AI

The LyricSense - Song Mood Classifier is a Natural Language Processing (NLP) project that analyzes song lyrics and predicts the emotional mood of the song.

The system takes lyrics as input and classifies them into moods such as:Happy, Sad, Romantic, Energetic
This project demonstrates how machine learning can understand text patterns and infer emotions from natural language.

## Features
    Predicts mood from user-input lyrics
    
    Fast and efficient (no heavy deep learning models)
    
    Uses TF-IDF + Machine Learning
    
    Interactive web app using Streamlit
    
    Displays clear prediction output

## Tech Stack

  Python 
  
  Pandas (Data Handling)
  
  Scikit-learn (ML Model)
  
  TF-IDF (Text Vectorization)
  
  Streamlit (Web App UI)
  
  Joblib (Model Saving)

## Dataset

  Dataset used:
  Mood-Based English Songs Dataset (Kaggle)
  
  It contains:
  
  Song lyrics
  Corresponding mood labels
